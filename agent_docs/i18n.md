# Translations (i18n)

Translation files: `src/locales/en.json` and `src/locales/he.json`
Context: `src/contexts/Translation/translation.context.tsx`

## Adding New Translations

Add keys to BOTH `en.json` and `he.json`:

```json
// en.json
{ "articleDetail": { "share": "Share", "linkCopied": "Link copied to clipboard!" } }

// he.json
{ "articleDetail": { "share": "שתף", "linkCopied": "הקישור הועתק!" } }
```

## Usage

```typescript
const { t, locale } = useTranslation();

<button>{t("articleDetail.share")}</button>
```

## Rules

- NEVER hardcode Hebrew or English strings — always use `t()` keys
- Always add keys to both language files simultaneously
- The codebase currently has ~30 files with hardcoded Hebrew strings that should be migrated
